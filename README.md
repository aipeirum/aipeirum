## What Apeirum is building

- Private AI systems for knowledge-heavy enterprise workflows
- A native Brazilian Portuguese model pipeline (`ptbr-llm`)
- Retrieval and evaluation layers for grounded answers
- A Rust inference core designed for local execution

## Current status

- Training and evaluating `ptbr-llm`
- Preparing SFT and retrieval datasets
- Building the first product and admin infrastructure
- Pre-launch

## Principles

- Privacy by design

- Reproducible evaluation
- Engineering-first execution
